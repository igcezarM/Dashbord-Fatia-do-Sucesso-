# Dashbord Fatia do Sucesso 🍕
🎯 Objetivo: Analisar vendas de pizzaria para identificar padrões de consumo e sabores mais populares.  📊 Análises SQL: Receita total 💰, quantidade vendida 📦, ticket médio 🎫, pedidos por dia/mês 📅, participação percentual por pizza 📈 e top 5 sabores mais vendidos 🏆.  💡 Insights: Ticket médio calculado ✅, dia da semana com maior movimento 📆, sazonalidade mensal 🌙 e ranking de faturamento por sabor 🍕.  🛠️ Ferramentas: SQL Server (extração/agregação) 🗄️, Power BI (dashboard interativo) 📊, Excel (base inicial) 📁.  ✨ Solução: Dashboard com filtros por período ⏰, tamanho 📏 e categoria 🏷️, permitindo análise de tendências 📉 e identificação dos produtos mais rentáveis 💎.
